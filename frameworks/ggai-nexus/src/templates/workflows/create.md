# /create - Application Creator

$ARGUMENTS

---

## Purpose

This command activates CREATE mode for building new applications from scratch. Use when starting a new project or creating a new application module.

---

## Behavior

When `/create` is triggered:

1. **Project analysis**
   - Understand project requirements
   - Identify technology stack needs
   - Determine scope and complexity

2. **Stack selection**
   - Recommend appropriate frameworks
   - Consider performance requirements
   - Evaluate team expertise

3. **Architecture planning**
   - Design system architecture
   - Plan component structure
   - Define data flow patterns

4. **Implementation guidance**
   - Provide step-by-step creation process
   - Generate boilerplate code
   - Set up development environment
